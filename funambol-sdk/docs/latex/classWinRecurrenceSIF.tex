\subsection{Detailed Description}
Rapresents a recurrence pattern object for Windows Clients, for SIF format. 

The object can be filled passing a SIF rec pattern string, or filling directly the map. Calling 'to\-String()' a SIF rec pattern string is formatted and returned.
