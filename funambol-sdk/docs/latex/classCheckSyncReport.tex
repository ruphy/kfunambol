\subsection{Detailed Description}
This class encapsulates logging and checking of a \doxyref{Sync\-Report}{p.}{classSyncReport}. 

When constructed with default parameters, no checking will be done. Otherwise the sync report has to contain exactly the expected result. When multiple sync sources are active, {\bf all} of them have to behave alike (which is how the tests are constructed).

No item is ever supposed to fail.
