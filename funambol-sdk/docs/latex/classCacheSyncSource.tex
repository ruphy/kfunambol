\subsection{Detailed Description}
This is an extension of the basic \doxyref{Sync\-Source}{p.}{classSyncSource} that a Sync\-ML client developer could implement to let the sync engine access the client's data. 

It provides the logic to retrieve the item modification to be exchanged with the server
